# Reverse DNS Lookup
    This is the Simple Python Program that will get the DNS Name Based on provided IP Address.

## Usage
Follow the Given Steps to Setup in Your Local System. 
- Create the Virtual Environment (Not Necessary but Recommand) and activate it.
    ```bash
        # For Windows User
        python -m venv venv
        venv\Scripts\activate

        #For Linux User
        python3 -m virtualenv venv
        source venv/bin/activate

    ```
- Install the Requirements
    ```bash
        pip install -r requirements.txt
    ```
- Run the `main.py` file
