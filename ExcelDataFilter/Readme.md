# Excel Data Filter
This is the Simple Python Program that will filter the Excel Data based on the given condition and save the Filtered Data into the New Excel File.  

## Usage
Follow the Given Steps to Setup in Your Local System. 
- Create the Virtual Environment (Not Necessary but Recommand) and activate it.
    ```bash
        # For Windows User
        python -m venv venv
        venv\Scripts\activate

        #For Linux User
        python3 -m virtualenv venv
        source venv/bin/activate

    ```
- Install the Requirements
    ```bash
        pip install -r requirements.txt
    ```
- Run the `main.py` file
